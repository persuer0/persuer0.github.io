具身智能强化营营员 · 2025.01.18 - 2025.02.28

🤖 技术方案：通过直接调用 API 访问 Qwen2-VL-72B-Instruct 实现 high-level 的 VLM 规划，下层部署了一个开源的 8自由度舵机小狗和一个使用乐高积木搭建的四轮差速小车。验证时，在办公桌上通过自编 UI 指定目标物体，两个硬件平台接收上层 VLM 输出的动作指令（标准自定义格式），通过正则解析后直接下发至客户端，实现了基于 VLM 的跨形态机器人统一控制。

🧠 RL 探索：在构建 8 自由度舵机小狗硬件平台过程中，进行了 URDF建模 → RL训练 → MuJoCo仿真 的 real2sim 工作，验证了强化学习控制舵机小狗的可行性，但因摩擦系数不合理和真机硬件执行精度偏差较大，未进行 RL 的 sim2real。

🔧 个人负责：两个真实物理世界的硬件平台搭建、客户端 ROS 控制系统搭建、RL 舵机四足狗的建模和 MuJoCo 环境参数配置。

研究计划

🔬 研究方向概述

经过两个多月的关键实习，我对 VLA（Vision-Language-Action）的研究产生了浓厚兴趣。当下 VLA 模型虽已具备一定智能性，但仍有大量进步空间，这让我看到了自己对这项技术有所贡献的希望。

我认为在具身智能蓬勃发展的时代，研究 VLA、不断推动其各方面性能提升是一件极具挑战且意义深远的事——很可能在短短几年内，VLA 或其他技术路线就能真正完美部署到真实物理世界中，并具备类人智能。

我对 VLA 模型的模型加速、泛化性提升、架构改进等方向都很感兴趣，研究生阶段非常期待参与相关工作。
